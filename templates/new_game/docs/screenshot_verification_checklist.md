# Screenshot Verification Checklist

Capture evidence after every function-skin, menu, HUD, overlay, result, or
settings reskin.

## Desktop viewport

- Viewport:
- Screenshot path:
- [ ] All labels fit inside their owner regions.
- [ ] Text does not overlap art, buttons, score, input fields, safe area, or
      neighboring controls.
- [ ] Screen still reads as a game screen.
- [ ] Theme art, gameplay language, and player feedback remain visible.
