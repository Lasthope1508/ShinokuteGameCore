# Agent Entry Gate

This repository contains Godot game projects. Do not start from memory.

Before changing, exporting, publishing, or deploying any game:

1. Identify the exact game project directory.
2. Read that project's local `AGENTS.md` if present.
3. Verify the source branch and commit from Git, not from a local assumption.
4. Use only files committed in Git unless the owner explicitly approves a local-only source.
5. Record branch, commit, export command, deploy target, and smoke result in the final report.

For Glyphflow Arrays, the project directory is:

```text
WaternetGodot_Cyberpunk
```

Its local instructions are mandatory:

```text
WaternetGodot_Cyberpunk/AGENTS.md
```
